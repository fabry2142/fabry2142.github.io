Dependency for using properly this page are Listed Here Marked >> #X#

>#1# Go to Openhab webpage on this link https://www.openhab.org/

Go to download section and choose your System

After Initialize Openhab open your Webpage on Localhost:8080

Consult the Documentation on this Link: https://www.openhab.org/docs/

>#2# Now you have to add new Binding Called Openhab Cloud Connector

>#3# Now you have to Register on this site https://myopenhab.org/

-1st Step is to insert your Email and Password

-2st Step is to insert the UUID and Secret Code from your Openhab Folder Files, Check Documentation for Help!
